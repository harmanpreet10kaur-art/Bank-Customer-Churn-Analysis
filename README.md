# 🏦 Bank Customer Churn Analysis

## 📌 Project Overview

This project analyzes customer churn in a banking environment using Power BI, SQL, Excel, Power Query, and DAX. The objective is to identify customer segments with a high probability of churn and provide actionable business recommendations to improve customer retention.

## 🚀 Project Objectives

- Analyze customer churn trends
- Identify high-risk customer segments
- Build interactive Power BI dashboards
- Perform customer segmentation using SQL
- Generate business insights and recommendations

## 🛠 Tools & Technologies

- Power BI
- SQL
- Microsoft Excel
- Power Query
- DAX

## 📂 Dataset

- 7 Integrated Banking Tables
- 10,000 Customer Records

Tables:
- CustomerInfo
- Bank Churn
- Active Customer
- Credit Card
- Geography
- Gender
- Exit Customer

## 📊 Dashboard Highlights

- Executive Dashboard
- Customer Demographics Dashboard
- Product Analysis Dashboard

KPIs:
- Overall Churn Rate
- Active Customers
- Exited Customers
- Average Balance
- Credit Score
- Customer Distribution

## 📈 Key Insights

- Overall churn rate: **20.37%**
- Germany recorded the highest churn rate (**32.44%**)
- Customers aged **46–55** had the highest churn (**50.57%**)
- Inactive members were significantly more likely to churn
- Credit card ownership showed minimal impact on retention

## 💡 Recommendations

- Improve retention for high-risk customer segments.
- Increase engagement among inactive customers.
- Launch targeted campaigns in Germany.
- Expand personalized loyalty programs.
- Use predictive analytics for proactive retention.
